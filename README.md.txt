# Java Swing Login & Welcome App

## Description
Ye simple Java Swing application hai jo **Login screen aur Welcome screen** demonstrate karta hai.  
- Username aur Password enter karne ke baad user ko Welcome screen dikhayi jati hai.  
- Username: `student`  
- Password: `1234`

## Features
- Simple Login validation  
- Welcome screen on successful login  
- Easy to understand code for beginners  

## How to Run
1. Java JDK install ho (version 8 ya upar)  
2. Dono files download karein:  
   - `LoginScreen.java`  
   - `WelcomeScreen.java`  
3. Terminal/Command Prompt me run karein:  
   ```bash
   javac LoginScreen.java WelcomeScreen.java
   java LoginScreen
